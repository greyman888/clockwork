# Clockwork

Clockwork is a Windows-first internal time tracking application for recording
time against consulting projects. It is built on a reusable
entity/component-based SQLite foundation so the application can evolve through
data definitions rather than repeated schema migrations.

For the main project, architecture, and product requirements, start with
[docs/PROJECT_GUIDE.md](docs/PROJECT_GUIDE.md). For layout and visual review,
see [docs/UI_DESIGN_GUIDE.md](docs/UI_DESIGN_GUIDE.md). For packaging and the
installer workflow, see [docs/WINDOWS_RELEASE.md](docs/WINDOWS_RELEASE.md).

If Codex skills are available, start a Clockwork coding session with
`$clockwork-project-guide`. That skill tells the agent to read
`docs/PROJECT_GUIDE.md` first and then pull in the layout or release guides
only when the task needs them.
